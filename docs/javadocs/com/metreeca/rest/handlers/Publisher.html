      • variants

        public static Stream<String> variants(String url)
        Computes HTML variants of a URL.

        Generate alternative paths appending .html/{index.html} suffixes as suggested by the path, e.g.:

        • …/path/ ›› …/path/, …/path/index.html
        • …/path/file ›› …/path/file, …/path/file.html

        URL anchors (#<anchors> are properly preserved.

        Parameters:
        url - the url whose variants are to be computed
        Returns:
        a stream of url variants
        Throws:
        NullPointerException - if url is null
